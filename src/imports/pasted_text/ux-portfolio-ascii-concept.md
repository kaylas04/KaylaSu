Create a personal UX design portfolio homepage inspired by the uploaded screenshots, but blend the visual language with subtle ASCII art aesthetics.

Goal:
Design a portfolio homepage for a UX designer that feels modern, refined, editorial, and slightly experimental. The overall site should still feel clean and recruiter-friendly, but with tasteful ASCII-inspired visual details that give it a unique digital identity.

Core concept:
Blend two directions:
1. A polished, spacious, modern portfolio style inspired by the uploaded screenshots
2. A subtle ASCII / terminal / computational aesthetic used as a visual accent, not as the dominant style

Important:
This should NOT feel like a hacker website, gaming site, or overly retro terminal interface.
It should feel like a sophisticated UX designer portfolio with a conceptual digital layer.

Overall mood:
- Clean and minimal
- Calm, intelligent, and intentional
- Slightly experimental
- Digital but elegant
- Editorial and structured
- Artistic without sacrificing usability
- Premium portfolio feeling

Use the uploaded screenshots as reference for:
- layout rhythm
- typography hierarchy
- spacing
- card structure
- section composition
- overall polish

Do not copy the screenshots literally.
Reinterpret them into a personal portfolio with a more distinctive identity.

Visual direction:
- Neutral or off-white background, or very light gray background
- Dark charcoal or near-black text
- One restrained accent color only, such as muted blue-gray, soft green, or pale silver
- Strong editorial typography hierarchy
- Use a clean sans-serif for primary text
- Use a monospace font selectively for ASCII-inspired labels, microcopy, dividers, tags, or decorative elements
- Spacious layout with strong margins and clear grid structure
- Refined, restrained rounded corners if needed

ASCII integration:
Use ASCII aesthetics in a subtle and elevated way. Examples:
- a hero background detail made of soft ASCII character patterns
- section dividers or framing elements built from monospace symbols
- project hover states with light ASCII-texture overlay
- small terminal-inspired labels or metadata styling
- decorative text-based grids, interface markers, or abstract symbol compositions
- minimal code/console-inspired micro details

ASCII should feel:
- abstract
- graphic
- intentional
- quiet
- integrated into the design system

Avoid:
- heavy black terminal screens
- neon hacker visuals
- dense illegible character blocks
- visual clutter
- gimmicky cyberpunk effects
- overly retro 1990s web styling
- making the whole site monospace
- sacrificing readability for style

Homepage structure:
1. Hero section
- Large headline introducing the designer
- Short subheading describing the designer as a UX designer interested in psychology, product thinking, and visual storytelling
- Primary CTA: View Work
- Secondary CTA: About / Resume / Contact
- Add a subtle ASCII-inspired visual layer in the hero background or side composition

2. Featured work section
- 3 selected project cards
- Each card should include title, short summary, role, and visual thumbnail
- Cards should feel refined, modern, and easy to scan
- Hover states may include subtle ASCII-inspired motion or texture

3. About / approach section
- Explain design philosophy in a concise way
- Emphasize clarity, human-centered thinking, thoughtful systems, and visual communication
- Could include a small ASCII-framed text block or structured monospace annotation

4. Contact / footer section
- Minimal and clean
- Include space for email, LinkedIn, and resume
- Optional subtle ASCII divider or signature element
- No fake agency/footer template elements

Motion direction:
Design with subtle motion in mind:
- soft fade and slide entrances
- elegant hover states
- gentle transitions
- motion should support hierarchy and atmosphere
- ASCII-inspired elements can animate lightly, but should never distract from content

Design principles:
- Work should remain the focus
- Visual style should feel distinct but professional
- The site should feel like a real UX portfolio, not a concept poster
- Keep the balance between usability and artistic identity
- Make it feel custom, thoughtful, and portfolio-ready

Final impression:
A modern UX portfolio homepage that feels polished and calm at first glance, but reveals subtle ASCII-inspired digital textures and details upon closer look.
